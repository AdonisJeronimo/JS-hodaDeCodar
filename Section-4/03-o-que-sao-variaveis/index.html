Como salvar valores

  • Salvamos os valores em variaveis(isso é um statement);
  • let laranjas = 5;
  • Depois podemos criar expressoes com os valores salvos;
  • console.log( laranjas * laranjas)
  • console.log( `Existem ${laranjas} laranjas na sexta`)
  • Podemos também mudar o valor que foi salvo anteriormente;
  • laranjas = larajnas + 1;
  • um let pode definir varias variaveis
  • let quadrado = 4, triangulo = 3, retangulo = 2.
