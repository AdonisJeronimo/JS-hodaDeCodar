A estrutura de uma função

Antes de mais nada, funções são chamadas durante o programa e você também pode ouvir alguem dizendo para executar, chamar e invocar uma função.

Para chamarmos uma função, utilizamos o nome dela e abrimos e fechamos parenteses. Dentro do parenteses podemos insirir parametros(em alguns casos é obrigatorio) que são basicamente valores que podem mudar o resultado da função.

function nomeDaFuncao();
function nomeDaFuncao (parametro1,parametro2);
